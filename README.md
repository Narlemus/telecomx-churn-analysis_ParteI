📊 TelecomX - Churn Analysis

📌 Descripción

Proyecto de análisis exploratorio de datos (EDA) enfocado en identificar patrones relacionados con la evasión de clientes (Churn) en TelecomX, usando Python

El objetivo es entender qué factores influyen en la cancelación del servicio y generar insights que ayuden a mejorar la retención de clientes.

🛠 Tecnologías utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter / Google Colab

📈 Análisis realizado

En este proyecto se realizaron:

* Limpieza y preparación de datos
* Análisis exploratorio de variables clave
* Visualización de patrones de churn
* Análisis de correlación entre variables

Se analizaron variables como:
* Cargo mensual
* Tiempo de permanencia (Tenure)
* Tipo de contrato
* Cuentas diarias

🔎 Principales insights

* Los clientes con contratos mensuales presentan mayor probabilidad de churn.
* Los clientes con menor tiempo de permanencia tienen mayor tasa de cancelación.
* Los cargos mensuales más altos se asocian con mayor evasión.

💡 Recomendaciones

* Incentivar contratos de mayor duración.
* Implementar estrategias de retención para clientes nuevos.
* Evaluar planes con cargos elevados para mejorar su valor percibido.
